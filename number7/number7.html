Sketch #7

The inequity this system models is the same number of clicks produces a different outcome. Repetition makes the green bar go up everyclick but the same amount of clicks takes the red bar longer to fill up. I wanted to add on to this sketch a time element to the red bar side because to add stress, I don't like how quickly it shows up and I wish it had a count down but I couldn't figure it out and I didn't want it to get too complicated. I learned the priviledged function and makes me think how I can implement in my upcoming sketches.
